
# 👟SHOES-MALL
> **KREAM|한정판 스니커즈 거래 플랫폼** 클론 프로젝트

<br/>

## 프로젝트 개요 

### 프로젝트 목적 

- KREAM 클론 프로젝트를 통하여 JPA와 REST API의 학습 목적으로 개발됨
- 세팅부터 개발까지 스스로 하면서 해당 지식 습득과 개발 역량 강화

### 프로젝트 개발환경

```
• IDE : IntelliJ IDEA Ultimate
• 언어 : Java 8
• 프레임워크 : SpringBoot 2.6.13
• 빌드도구 : Gradle 
• 데이터베이스 : Maria DB
```
